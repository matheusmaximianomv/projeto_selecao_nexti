Installing Git for Windows

On the official Git website click on download for Windows and on the next page choose the type of your system, whether it is 32 or 64 bit. Continue to press "Next" until the installation is complete. In the Windows version, Git comes with its own command prompt (Git Bash), which in addition to the git commands also provides some Unix commands that can be very handy (as well as being much nicer than the standard Windows command prompt ).

Creating a New Repository on Github

On the official GitHub website, sign up and after your email is authenticated, point your cursor to the upper right corner and click on the plus icon and select the "New Repository" option. You will then be redirected to a page where you will create the new repository by defining the name, its description, its type of visibility - whether public or private - and the option to generate a README.md file, to be displayed on the home page from the project.

SSH Authentication

After signing up for GitHub, login, and now we need to create an SSH key. In any directory of your operating system, right-click and select the "Git Bash Here" option, after opening enter the following command:

ssh-keygen -t rsa -C "your_mail@email.com"

Enter your email address in GitHub. Press enter on the next question, leaving the default file name to be generated. The next question will ask you for a password. If you wish, create the password, if not, press enter. Enter now: 

notepad ~/.ssh/id_rsa.pub

The key will appear in Notepad, so you can copy it.

Now go to the upper right corner, click on your GitHub user, go to "Settings" and then "SSH and GPG Keys". Click "New SSH key." Enter a title for the key. And paste in the "Key" field the entire contents of the id_rsa.pub file, which was opened in Notepad.

Organizing the House

At the end of the SSH configuration, we can start using Git. First add the username and email, remembering that the email has to be registered in GitHub, with the following commands:

git config --global user.name "First Name Last Name"
git config --global user.email "your_email@email.com"

After executing the commands, we will initialize a Git repository in the folder we are:

git init

As an example, we will create a .txt file to add together with other possible files already in your directory to a commit:

touch test.txt
git add .
git commit -m "First commit"

Still in "Git Bash" enter the command to add the remote GitHub repository to your project:

git remote add origin https://github.com/user/repository.git

Finally, to send your project changes to your GitHub repository, type:

git push origin master

By following these steps you will have learned how to use Git and GitHub for the first time. For more information go to the Git or GitHub documentation website. Good Studies!

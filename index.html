What's Git?

Git is a distributed version control system, used mainly in software development, but can be used to record the history of edits of any type of file. It is currently free software, distributed under the terms of version 2 of the GNU General Public License. Its maintenance is currently supervised by June Hamano.

Git was created by Linus Torvalds, famous for his other creations, Linux. He needed something relatively fast that could deal with a series, almost constant, of activities involved in something like Linux, having the same to be free. Finding nothing, he decided to create Git. As in the case of Linux, Torvalds baptized him in reference to himself; in British English, Git is a slang for "hardheaded, who thinks he's always right, quarrelsome."

Linus Torvalds

Another important factor of Git is the possibility to create, at any moment, several snapshots of your project or branch. Suppose your project is an HTML site, and you want to create a new section in your HTML code, but at that point you do not want these changes to be available to anyone else, just for you. That is, you want to change the project (including multiple files in it), but you still do not want this to be treated as "official" to other people, so you create a branch (like a mirror copy) , until you hit all the details of it. After that, you can merge back from your branch to the original project.

What's Github?

GitHub is a source code hosting platform with version control using Git. It allows programmers, utilities, or any user on the platform to contribute to private and / or Open Source projects from anywhere in the world. GitHub is widely used by programmers to publicize their work or for other programmers to contribute to the project, as well as to promote easy communication through resources that report problems or merge remote requests (pull requests).

GitHub is globally used and has over 3 million active users worldwide contributing to business or personal projects. Today GitHub houses more than 65 million projects, some of which are known worldwide. WordPress, GNU / Linux, Atom, Electron. GitHub also supports the organization feature that is widely used by those who want a larger scale for their projects. Most of the time, the feature is used by already existing companies like Google, Microsoft and WordPress.

On June 4, 2018, Microsoft announced the purchase of the platform for US $ 7.5 billion, equivalent to R $ 27.225 billion with US dollar at "3.63" 06/06/2018. Satya Nadella, Microsoft's chief executive, reaffirms Microsoft's new stance on open source, saying: "Microsoft is a company that develops in the foreground and, by joining forces with GitHub, we strengthen our commitment to freedom, openness and developer innovation.We recognize the responsibility of the community we assume with this agreement and we will do our best to enable each developer to build, innovate and solve the most pressing challenges in the world.
